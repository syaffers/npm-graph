Graph of the Top 50 Most Depended-on Node Libraries and its dependencies

Nodes and Edges (Ovals and Arrows):

Nodes represent a package and a directed edge (arrow) points to the package on which it depends. Blue nodes are packages belonging to the "Top 50" list, yellow nodes are packages it depends on.
